### 5. 测试一键修复功能

#### 5.1 测试场景：环境变量冲突
**前置条件**：
```bash
# 创建环境变量冲突
export ANTHROPIC_API_KEY="test_key_conflict"
echo 'export ANTHROPIC_API_KEY="test_key_conflict"' >> ~/.zshrc
```

**操作步骤**：
1. 重启应用或重新诊断
2. 应该看到"环境变量冲突：ANTHROPIC_API_KEY"问题
3. 点击"一键修复"按钮

**预期结果**：
- [ ] 按钮显示"修复中..."加载状态
- [ ] 自动备份并删除冲突的环境变量
- [ ] 修复成功后显示 toast 提示："成功修复 1 个问题"
- [ ] 自动重新诊断，冲突消失

**验证修复**：
```bash
# 检查环境变量是否已删除
echo $ANTHROPIC_API_KEY  # 应该为空
grep ANTHROPIC_API_KEY ~/.zshrc  # 应该找不到
```

**恢复**：
```bash
# 清理测试数据
unset ANTHROPIC_API_KEY
sed -i '' '/ANTHROPIC_API_KEY/d' ~/.zshrc
```

#### 5.2 测试场景：配置文件损坏
**前置条件**：

**操作步骤**：

3. 点击"一键修复"按钮

**预期结果**：
- [ ] 自动从备份恢复或生成默认配置

**验证修复**：
```bash

```
