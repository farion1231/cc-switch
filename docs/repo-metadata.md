# Repository Metadata Suggestions

This file keeps the repository naming and release wording consistent across GitHub pages and README files.

## Recommended Project Name

`CC Switch Legacy`

## Recommended GitHub Description

### English

`A compatibility-focused fork of CC Switch with macOS 10.15 Catalina support.`

### Chinese

`CC Switch 的兼容性分支，专门为 macOS 10.15 Catalina 提供支持。`

## Recommended Release Title Format

Use this format for GitHub releases:

```text
v{version} - CC Switch Legacy
```

If you want the compatibility target to be more explicit:

```text
v{version} - CC Switch Legacy (macOS 10.15 Compatible)
```
